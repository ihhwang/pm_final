# [SmartPlug]_1.2_RACI

# RACI 정의

- **R**esponsible / 실제 수행자 (실행 책임), 최소 1개의 R
- **A**ccountable / 최종 의사결정 및 결과 책임 (1명 원칙), 반드시 단일 A

  - 의사 결정자와 결과 책임자가 분리되는 경우, 결과를 책임지는 자를 A로 지정하고 의사결정자는 C\*로 표기함.
- **C**onsulted / 사전 협의 대상 (양방향 커뮤니케이션)
- **I**nformed / 결과 통보 대상 (단방향 커뮤니케이션)
