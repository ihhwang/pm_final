# [SmartPlug]_10.3_조달 요구사항 명세서

# 1. 조달 요구사항 명세

| ID | 요구사항 | 검증 방법 | 결과 |
| --- | --- | --- | --- |
| PROC\_REQ\_001 | 기본 부팅 정상 동작 | 전원 인가 후 부팅 완료 확인 | 충족 |
| PROC\_REQ\_002 | 마이크 입력 기본 동작 | 마이크 입력 신호 수집 동작 확인 | 충족 |
| PROC\_REQ\_003 | 스피커 출력 기본 동작 | 스피커 출력 재생 동작 확인 | 충족 |
| PROC\_REQ\_004 | Wi-Fi 기본 연결 동작 | AP 접속 및 통신 동작 확인 | 충족 |
| PROC\_REQ\_005 | 220V 직결 전원부 안전 확보 | 전원부 안전 기준 충족 확인 | 충족 |
| PROC\_REQ\_006 | 최소 2EA 제공 | 수량 확인 | 충족 |

- 요구사항 명세는 HW PL에 이메일로 전달, 협의 (2025년 7월 1일)
