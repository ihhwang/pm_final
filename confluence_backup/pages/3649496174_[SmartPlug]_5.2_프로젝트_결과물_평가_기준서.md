# [SmartPlug]_5.2_프로젝트 결과물 평가 기준서

# 1. 평가 항목표 / 기준 / 관리 방법

| 항목 | 평가 기준 | 평가 방법 | 승인 기준 |
| --- | --- | --- | --- |
| 요구사항 | 각 요구사항 만족 여부 | 문서 리뷰 | 요구사항 100% 반영 |
| 기능 적합성 | 세부 기능 구현 완료 여부 | DoD, 기능 테스트 결과 문서 리뷰, 잔여 이슈 리뷰 | 테스트 결과 100%, A·B급 잔여 이슈 0 |
| 품질 | 통합 테스트 완료 여부 | 통합 테스트 결과 문서 리뷰 | 테스트 결과 100% |
| 성능 | 성능 테스트 완료 여부 | 성능 테스트 결과 문서 리뷰 | 시연 시나리오 기준 네트워크 처리율 및 지연 수준 만족 |
| 문서화 | 문서화 산출물 작성 및 업데이트 여부 | Collab 업데이트 여부 확인 및 해당 문서 리뷰 | 산출물 업데이트 100% |
| 산출물 버전 관리 | 최신 버전 관리 여부 | 최종 업데이트 시점 확인 | 점검 시점 기준 최대 1일 이내 |

# 2. 평가 데이터 관리 방법

- 주요 마일스톤 시점에 위 평가 항목 전체를 기준으로 평가 진행
- 평가 결과 데이터는 Collab을 통해 관리하며, 마일스톤별 상세 기록은 [5.3 프로젝트 결과물 평가 수행서](/main/spaces/~irene.hwang/pages/3649496178/SmartPlug+_5.3_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8+%EA%B2%B0%EA%B3%BC%EB%AC%BC+%ED%8F%89%EA%B0%80+%EC%88%98%ED%96%89%EC%84%9C)에 기록
- 평가는 TL(팀장) 또는 iLab장에 보고하여 확인·승인을 받는 방식으로 진행
