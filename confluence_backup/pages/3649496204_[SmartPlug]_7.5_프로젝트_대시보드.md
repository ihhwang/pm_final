# 3. 일정 편차 분석

## 3.1 분석 기준

| 구분 | 내용 |
| --- | --- |
| 중간 기준점 (V1) | CR-01/02 발생 직전 시점 (2025.08.01) — 원래 기준선 적용 (WBS 61개) |
| 최종 기준점 (V2) | 과제 종료 시점 (2025.12.09) — 변경 후 기준선 적용 (WBS 64개, 종료일 12/01) |
| 측정 지표 | WBS 완료 개수 기준 진척률 (기준선별 전체 WBS 수 대비) |

## 3.2 V1 - CR-01/02 발생 직전 (2025.08.01, 원래 기준선)

| 항목 | 계획 | 실적 | 편차 |
| --- | --- | --- | --- |
| WBS 완료 수 | 12개 / 61개 (19.7%) | 15개 / 61개 (24.6%) | +3개 (계획 대비 선행) |

- HUB 개발자(윤, 홍) 합류 전으로 PLG WBS 중심으로 진척된 시점임
- PLG 15개 완료 (계획 12개 대비 3개 선행) - 초기 일정 계획이 현실적으로 수립되었음을 확인. CR-01/02 발생 후 대응 여력 확보에 기여함

## 3.3 V2 - 과제 종료 (2025.12.09, 변경 후 기준선)

| 항목 | 기준선 | 실적 | 편차 |
| --- | --- | --- | --- |
| 종료일 | 2025.12.01 (CR-01/02 이후 조정) | 2025.12.09 | +8일 지연 |
| WBS 완료 수 | 64개 / 64개 (100%) | 61개 / 64개 (95.3%) | -3개 미완료 |

- 8일 지연 원인: CR-01/02 이후 신규 기능(인터컴·멀티카메라·멀티플러그) 통합 마무리에 집중함으로써 발생한 후속 처리 지연
- 미완료 3개 항목: 멀티플러그 확장을 위한 기능 고도화 관련 항목 → 시연 핵심 범위 외 항목으로 프로젝트 목표(시연 완료) 달성에 영향 없음
- 핵심 시연 목표(음성제어·인터컴·멀티카메라 통합 시나리오) 100% 달성

## 3.4 종합 평가

| 평가 항목 | 결과 |
| --- | --- |
| 변경 전 일정 관리 | 계획 대비 선행 (WBS 3개 조기 완료) - CR 발생 시 대응 여력 확보에 기여 |
| 변경 영향 흡수 | 기준선 +14일 조정 후 8일 지연으로 마무리 - 추가 인력 투입없이 기존 4인으로 대응 완료 |
| 목표 달성 수준 | 시연 시나리오 100% 완료, WBS 전체 95.3% 완료 (미완료 3개 : 시연 외 고도화 항목) |
