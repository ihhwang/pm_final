# 1. 프로젝트 개요

|  |  |
| --- | --- |
| 과제명 | On-Device AI Hub를 위한 스마트 플러그 PoC 개발 |
| 목표 | 단일 홈 허브가 주택/건물 내에서 실제로 사용 가능한 물리적 공간 확장을 가능하게 하는 악세서리 기기의 PoC SW를 개발하고 이를 On-Device AI Hub와 연동하여 최소한의 실제 동작을 시연함 |
| 기간 | 2025.04.01 ~ 2025.12.1 |
| 인원 | 총 4명 / 23MM |
| 고객 가치 | 홈 허브의 위치를 옮기거나, 여러개의 허브를 구입하지 않고 간단한 악세서리를 추가함으로써 집 안 어디서나 홈 허브의 기능을 사용할 수 있다 |
| 사업적 가치 | 본 과제는 PoC 목적의 과제로 사업적 가치를 직접적으로 언급하기는 무리가 있으나, 향후 홈 허브 사업이 일정 규모를 이루게 되는 경우 이 과제를 확장함으로써 다음을 달성할 수 있다.   - 홈 허브 패키지의 가격 경쟁력 확보, 시장 확대 가속화 - 분산형 AI 홈허브 인프라 구축을 통한 AI 인프라 비용 절감 - ThinQ 가전 연계를 통한 ThinQ 생태계 확대 |
| 과제 진행 관리 | Collab / Waterfall 방식 진행 |

현실적으로 프로젝트 팀 구성원 다수가 Agile 개발방법론 경험 부족으로 Agile 도입이 사실상 불가능한 상황이었음 - 향후 유사한 상황에서 프로젝트 팀을 어떻게 Agile 방식으로 연착륙 시킬 것인지에 대해 고민이 필요함
