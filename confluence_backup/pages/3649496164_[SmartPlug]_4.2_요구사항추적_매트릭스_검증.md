# [SmartPlug]_4.2_요구사항추적 매트릭스 검증

본 매트릭스는 기능 요구사항 및 비기능 요구사항이 WBS 작업 항목에 연결/구현되어 있음을 추적, 검증하기 위한 문서임

V&V(Verification & Validation) 관점에서 각 요구사항이 어떤 작업으로 구현되고 검증되었는지 확인함.

# 1. 표 정보

| 구성 요소 | 내용 |
| --- | --- |
| **행 (Row)** | 요구사항 ID |
| **열 (Column)** | WBS 작업 ID |
| **셀 표시** | x -  해당 WBS 작업이 해당 요구사항을 구현/검증  (공백) - 해당되지 않음 |

# 2. 주요 현황

| 항목 | 내용 |
| --- | --- |
| 추적 요구사항 | 총 47개 (기능 44개, 비기능 3개) |
| 연결 WBS 작업 | 65개 (CR-01/02 변경 이후 추가 WBS 포함) |
| 누락 요구사항 | 없음 - 전체 요구사항에 1개 이상의 WBS 작업 연결 확인 |
| 변경 반영 | CR-01(인터컴), CR-02(다중 플러그) 이후 추가된 WBS 항목 매트릭스 반영 완료 |

※ 본 과제는 PoC 목적으로 공식 테스트 케이스 문서를 별도 작성하지 않음.

   구현 완료 여부는 WBS DoD(Definition of Done) 기준으로 확인, 검증 결과는 [5.3 결과물 평가 수행서](/main/spaces/~irene.hwang/pages/3649496178/SmartPlug+_5.3_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8+%EA%B2%B0%EA%B3%BC%EB%AC%BC+%ED%8F%89%EA%B0%80+%EC%88%98%ED%96%89%EC%84%9C)에 마일스톤별로 기록함
