# [SmartPlug]_10_조달관리

# 1. 조달 계획

|  |  |
| --- | --- |
| 조달 유형 | 내부 조달 (사내 HW 개발 태스크 협력, 공식 계약 없음) |
| 조달 목적 | SmartPlug PoC 개발을 위한 IoTPlug B/D(개발용 보드) 2EA 확보 |
| 공급자 | On-Device AI Hub HW 개발 태스크 (HW PL) |
| 조달 품목 | IoT SmartPlug 개발용 보드 |
| 인수 기준 | 기능별 인수 요구사항 5개 항목 (상세: 10.3 참조) |
| 조달 일정 | 2025년 7월 1일 : 인수 요구사항 전달  2025년 8월 첫째 주 : 시료 수령  2025년 8월 : 인수 검증 및 피드백 |
| 특이사항 | 내부 조달 특성상 공식 계약서가 아닌, 이메일을 통한 요구사항 협의 및 확인으로 대체 |

|  |  |

# 1. 조달 요구사항 명세

| ID | 요구사항 | 검증 방법 | 결과 |
| --- | --- | --- | --- |
| PROC\_REQ\_001 | 기본 부팅 정상 동작 | 전원 인가 후 부팅 완료 확인 | 충족 |
| PROC\_REQ\_002 | 마이크 입력 기본 동작 | 마이크 입력 신호 수집 동작 확인 | 충족 |
| PROC\_REQ\_003 | 스피커 출력 기본 동작 | 스피커 출력 재생 동작 확인 | 충족 |
| PROC\_REQ\_004 | Wi-Fi 기본 연결 동작 | AP 접속 및 통신 동작 확인 | 충족 |
| PROC\_REQ\_005 | 220V 직결 전원부 안전 확보 | 전원부 안전 기준 충족 확인 | 충족 |
| PROC\_REQ\_006 | 최소 2EA 제공 | 수량 확인 | 충족 |

- 요구사항 명세는 HW PL에 이메일로 전달, 협의 (2025년 7월 1일)
