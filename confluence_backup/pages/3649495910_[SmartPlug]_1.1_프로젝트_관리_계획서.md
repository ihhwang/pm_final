목차

# 1. 프로젝트 개요

|  |  |
| --- | --- |
| 과제명 | On-Device AI Hub를 위한 스마트 플러그 PoC 개발 |
| 목표 | 단일 홈 허브가 주택/건물 내에서 실제로 사용 가능한 물리적 공간 확장을 가능하게 하는 악세서리 기기의 SW를 개발하고 이를 On-Device AI Hub와 연동하여 최소한의 실제 동작을 시연함  악세서리 기기는 전원 공급의 용이성 및 음성 명령을 중심으로 한 기존 홈 허브의 사용성을 고려하여 전원 플러그 형태로 개발(이하 스마트 플러그 또는 플러그로 칭함)하는 것을 전제하며, On-Devce AI Hub HW 개발 Task에서 별도로 진행함.   - 전체 연동 구조 설계 및 개발 범위 확정 - 스마트 플러그 SW 개발   - 스마트 플러그 요구사항 발굴 및 확정   - 스마트 플러그 기동 SW 구현 및 검증 - On-Device AI Hub 측 SW 개발   - 스마트 플러그 연동을 위한 인터페이스 SW 개발   - On-Device AI Hub와 AI 연동 SW개발 - 스마트 플러그 연동 시연   - 시연 시나리오 발굴 및 확정   - 통합 연동 테스트 및 안정화 |
| 기간 | 2025.04.01 ~ 2025.11.30 |
| 인원 | 총 4명 / 23MM |
| 고객 가치 | 홈 허브의 위치를 옮기거나, 여러개의 허브를 구입하지 않고 간단한 악세서리를 추가함으로써 집 안 어디서나 홈 허브의 기능을 사용할 수 있다 |
| 사업적 가치 | 본 과제는 PoC 목적의 과제로 사업적 가치를 직접적으로 언급하기는 무리가 있으나, 향후 홈 허브 사업이 일정 규모를 이루게 되는 경우 이 과제를 확장함으로써 다음을 달성할 수 있다.   - 홈 허브 패키지의 가격 경쟁력 확보, 시장 확대 가속화 - 분산형 AI 홈허브 인프라 구축을 통한 AI 인프라 비용 절감 - ThinQ 가전 연계를 통한 ThinQ 생태계 확대 |
| 과제 진행 관리 | Collab / Waterfall 방식 진행 |

| 개발 범위 | - 전체 연동 구조 설계 - 스마트 플러그 요구사항 발굴   - 홈허브 연동   - 기동어 처리   - 음성 제어   - 인터콤 - 스마트 플러그 기동 SW 구현 - 스마트 플러그 기동 SW 검증 - On-Device AI Hub 측 플러그 연동 인터페이스 SW 개발 - On-Device AI Hub의 AI 연동 SW개발 - 스마트 플러그 연동 시연 시나리오 발굴   - 인터콤은 시연 범위제외, 스마트 홈 음성 제어만 대상으로 함   - 플러그-허브는 1:1 연결을 기준으로 함 |
