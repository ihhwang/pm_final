**v2 → v3 변경 요약**
- S2(흐름 요약) 삭제 → CR-01/02 "사건" 중심 슬라이드로 교체, 발표 전체의 spine으로 전진배치
- S4는 "통합관리 실행 디테일"로 역할 축소 (판단은 S2로 이동, 실행은 S4에 유지)

- S8.5(Self Assessment 표)는 채택하지 않음 — 심사위원이 매길 점수를 발표자가 선제적으로 낮춰버리는 위험 때문. 자기인식은 S9 근거 문장으로만 녹임
- 전 슬라이드 제목을 "내용 요약형"에서 "스토리 기능형"으로 교체

